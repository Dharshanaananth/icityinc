# iCity Inc Corporate Website

## Overview

The iCity Inc Corporate Website is a professional web platform designed to present the company's vision, services, business verticals, career opportunities, and contact information. The website provides visitors with a clear understanding of the organization while maintaining a modern and responsive user experience.

## Features

* Corporate landing page
* Company overview section
* About Us page
* Business verticals showcase
* Career opportunities page
* Contact information page
* Responsive website design
* Organized asset and resource management
* Lightweight and fast-loading static website

## Project Structure

```text
icityinc/
│
├── assets/          # Images, icons, and media resources
├── css/             # Stylesheets
├── js/              # JavaScript files
├── pages/           # Website pages
├── partials/        # Reusable HTML components
├── index.html       # Main entry point
└── package.json     # Project configuration
```

## Technologies Used

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/icity-inc-website.git
```

Navigate to the project directory:

```bash
cd icity-inc-website
```

Open the project:

```bash
index.html
```

or run a local server:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```
